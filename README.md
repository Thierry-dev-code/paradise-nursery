# Paradise Nursery Shopping Application

## Project Name: Paradise Nursery

## Description
Paradise Nursery is a dynamic e-commerce web application built with React and Redux. It allows users to browse a wide variety of houseplants, view plant details (image, name, description, price), add plants to a shopping cart, and manage their cart by adjusting quantities or removing items.

## Features
- Browse plants organized into categories
- Add plants to a shopping cart
- View and manage cart items
- Dynamic cart count displayed in the navbar
- Responsive and user-friendly design

## Technologies Used
- React (with Vite)
- Redux Toolkit
- CSS3

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open your browser at `http://localhost:5173`
